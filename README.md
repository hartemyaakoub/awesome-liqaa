## Official SDKs

- **[liqaa-js](https://github.com/hartemyaakoub/liqaa-js)** — JavaScript / TypeScript SDK + drop-in `<script>`. The flagship.
- **[liqaa-php](https://github.com/hartemyaakoub/liqaa-php)** — Server-side PHP client (Composer).
- **[liqaa-python](https://github.com/hartemyaakoub/liqaa-python)** — Pure-Python (PyPI).
- **[liqaa-go](https://github.com/hartemyaakoub/liqaa-go)** — Idiomatic Go module.
- **[liqaa-mcp](https://github.com/hartemyaakoub/liqaa-mcp)** — Model Context Protocol server (let AI agents start video calls).

## Specifications

- **[liqaa-openapi](https://github.com/hartemyaakoub/liqaa-openapi)** — OpenAPI 3.1 spec for the public API.
